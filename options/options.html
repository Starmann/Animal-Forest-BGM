Playback Settings
Music Volume
The volume that you would like the background music to play at, from 0 to 100. (Default volume is 30.)
Pause On Startup
Defaults the music playback to the paused state when you start your browser.
Silence During Media Playback
Automatically pauses and resumes music playback when media elements in a tab start or stop playing audio.
Fade On Transitions
Enables smooth fading between song changes every hour or (if enabled) when the weather changes. Might cause audio stuttering on slower machines.
Fade On Pause/Resume
Enables smooth fading when the current song is paused or unpaused. Might cause audio stuttering on slower machines.

Notification Settings
Show Notifications
Enables desktop notifications that pop up when the song changes every hour.
Notification Format
The time format to be used in desktop notifications, if enabled.

Appearance Settings
Toolbar Icon Style
If the default toolbar icon (Grey Icon) conflicts with the look of your browser, you can choose an alternative icon to display.
Pause Indicator Color
If the default color (Firefox Light Blue) of the pause indicator blends into the background of your browser too easily, you can change it here.

Other
Reset All Settings
Click the button to the right in order to reset all of the settings back to their defaults. (You will be prompted before the reset.)
